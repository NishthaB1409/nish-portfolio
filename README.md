# Nishtha Bhagyawant Portfolio

Reference-style AI/ML portfolio website for Nishtha Bhagyawant, an AI Engineer focused on LLMs, RAG pipelines, computer vision, FastAPI integration, and AWS deployment.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The app is built with React functional components, Vite, and Tailwind CSS.

## Structure

- `src/components` - reusable UI primitives
- `src/sections` - page sections
- `src/data/portfolio.js` - portfolio content and section data
